About

เว็บไซต์นี้เป็นเพียงโปรเจคส่วนตัวที่สร้างขึ้นเพื่อการเรียนรู้และทดลองทำเว็บไซต์ด้วยตัวเอง ไม่ได้มีจุดประสงค์ทางการค้าหรือเชิงพาณิชย์ใดๆ เป็นพื้นที่สำหรับการทดลองและพัฒนาทักษะด้านการเขียนโค้ด รวมถึงเป็นที่เก็บผลงานและบันทึกการเรียนรู้ต่างๆ

สร้างด้วย HTML, CSS และ JavaScript อย่างง่ายๆ เพื่อศึกษาการพัฒนาเว็บไซต์เบื้องต้น และทดลองการใช้งานเทคโนโลยีต่างๆ โดยเน้นการเรียนรู้จากการลงมือทำจริง การแก้ไขปัญหาที่เกิดขึ้น และการปรับปรุงพัฒนาอย่างต่อเนื่อง เพื่อให้เข้าใจหลักการทำงานของเว็บไซต์อย่างลึกซึ้ง

วัตถุประสงค์หลักคือการเรียนรู้และพัฒนาทักษะการเขียนโค้ด การออกแบบเว็บไซต์ และการทำความเข้าใจพื้นฐานของการพัฒนาเว็บไซต์ นอกจากนี้ยังมุ่งเน้นการศึกษาแนวทางการพัฒนาที่เป็นมาตรฐาน การจัดการโค้ดให้เป็นระเบียบ และการสร้างเว็บไซต์ที่มีประสิทธิภาพ

ในการพัฒนาเว็บไซต์นี้ ได้นำเอาหลักการออกแบบที่เน้นผู้ใช้เป็นศูนย์กลาง (User-Centered Design) มาประยุกต์ใช้ โดยคำนึงถึงความสะดวกในการใช้งาน ความเรียบง่ายของการออกแบบ และการตอบสนองที่รวดเร็ว รวมถึงการรองรับการแสดงผลบนอุปกรณ์ที่หลากหลาย (Responsive Design)

This website is a personal project created for learning and experimenting with web development. It has no commercial or business purposes. It serves as a space for experimentation and coding skill development, as well as a portfolio and learning journal.

Built with simple HTML, CSS, and JavaScript to study basic web development and experiment with various technologies. The focus is on learning through hands-on experience, problem-solving, and continuous improvement to gain a deep understanding of how websites work.

The main objective is to learn and develop coding skills, website design, and understand the fundamentals of web development. Additionally, it aims to study standard development practices, code organization, and efficient website creation.

In developing this website, user-centered design principles have been applied, focusing on usability, simplicity in design, and fast response times. The site also implements responsive design to ensure proper display across various devices.

Technologies Used:

HTML5 สำหรับโครงสร้างเว็บไซต์ HTML5 for website structure
Tailwind CSS สำหรับการจัดการสไตล์ Tailwind CSS for styling
JavaScript สำหรับการทำฟังก์ชัน JavaScript for functionality
GitHub Pages สำหรับการเผยแพร่เว็บไซต์ GitHub Pages for website hosting
Cloudflare สำหรับ DNS และระบบความปลอดภัย Cloudflare for DNS and security
